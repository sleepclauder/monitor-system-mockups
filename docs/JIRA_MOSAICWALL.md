# Мозаика (mosaicwall.html)

## Назначение

Страница для одновременного визуального контроля нескольких каналов вещания. Оператор сам формирует набор каналов на экране и сохраняет часто используемые раскладки.

## Поведение пользователя

### Сценарий 1: Собрать мозаику

1. Пользователь открывает страницу и видит пустую рабочую область с подсказкой "Перетащите канал из списка слева"
2. Слева — панель со списком всех каналов (название, статус, теги)
3. Пользователь находит нужный канал — через поиск по имени или фильтрацию по тегам (HLS / Live / MPEG-TS / VOD)
4. Перетаскивает канал из списка в рабочую область — появляется плеер с видеопотоком
5. Повторяет для других каналов — плееры автоматически выстраиваются в адаптивную сетку
6. Один и тот же канал можно добавить несколько раз (например, для сравнения)

### Сценарий 2: Управление плеерами

- **Навёл на плеер** — появляется заголовок с названием канала и кнопки управления
- **Удалить плеер** — кнопка `×` убирает конкретный плеер из сетки
- **Полный экран** — кнопка `⛶` разворачивает плеер на весь экран; закрыть — кнопка `✕` или Escape
- **Очистить всё** — кнопка в тулбаре убирает все плееры разом

### Сценарий 3: Сохранение и загрузка раскладок

1. Пользователь собрал нужный набор каналов
2. Нажимает "Закладки ▾" → вводит имя (например, "Утренний эфир") → "Сохранить"
3. В следующий раз — открывает "Закладки ▾", кликает по имени раскладки — мозаика восстанавливается
4. Ненужную раскладку можно удалить кнопкой `×` в списке

### Панель каналов

- Можно свернуть/развернуть кнопкой `◀` / `▶` чтобы освободить экран
- Фильтр по тегам — мультиселект: можно выбрать несколько тегов, покажутся каналы с любым из выбранных
- Кнопка "Все" сбрасывает фильтр
- У каждого канала цветная точка статуса: зелёная (online), оранжевая (warning), серая (offline)

## Ключевые правила

- Раскладки сохраняются локально в браузере (localStorage)
- Нельзя сохранить раскладку без имени или с пустой мозаикой
- Загрузка раскладки полностью заменяет текущий набор плееров
- Видеопотоки — визуальная имитация (градиенты с цветом канала)
