# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-02-07

### Добавлено
- Начальная версия мокапов системы мониторинга
- Страница Dashboard с общей статистикой и метриками
- Страница списка объектов мониторинга с фильтрами
- Детальная страница объекта с графиками и логами
- Страница алертов и уведомлений
- Страница отчетов и аналитики
- Страница настроек системы
- Темная тема оптимизированная для 24/7 работы
- Неоновые акценты (зеленый/синий)
- Анимации появления элементов
- Интерактивные hover-эффекты
- Единая навигация между страницами
- CSS переменные для легкой кастомизации
- README с полной документацией
- CONTRIBUTING руководство
- .gitignore для чистого репозитория
- index.html с автоматическим редиректом

### Технологии
- HTML5 с семантической разметкой
- CSS3 с flexbox/grid
- Google Fonts (Outfit + JetBrains Mono)
- CSS Animations
- CSS Custom Properties (переменные)

### Дизайн-система
- Основные цвета:
  - Фон: #0a0e1a, #111827, #1a2332
  - Акценты: #00ff88, #00ccff
  - Статусы: #ffaa00 (warning), #ff3366 (danger)
  - Текст: #e4e8f0, #8b94a8
- Типографика:
  - Основной шрифт: Outfit (300, 400, 600, 700)
  - Моноширинный: JetBrains Mono (400, 500, 700)
- Компоненты:
  - Карточки с border-radius: 12px
  - Кнопки с border-radius: 8px
  - Анимации: 0.2s - 0.5s ease
  - Hover эффекты с transform: translateY(-4px)

## Формат для будущих версий

### [Unreleased]
Будущие изменения, которые еще не выпущены

### [X.Y.Z] - YYYY-MM-DD
#### Добавлено (Added)
- Новая функциональность

#### Изменено (Changed)
- Изменения в существующей функциональности

#### Устарело (Deprecated)
- Функции, которые скоро будут удалены

#### Удалено (Removed)
- Удаленная функциональность

#### Исправлено (Fixed)
- Исправленные баги

#### Безопасность (Security)
- Исправления уязвимостей
