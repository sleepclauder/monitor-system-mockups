# Описание страницы настроек объектов мониторинга

Система расчитана на большое количество объектов, поэтому необходимо предусмотреть групповые настройки.
Предлагается использовать таблицу объектов (HLS или MPEG-TS потоков) для возможности применения групповых настроек.

# Групповые настройки объектов (Bulk Object Settings)

## Цель фичи

Позволить пользователю быстро применить одинаковые изменения к большому количеству объектов одновременно, минуя необходимость открывать каждую карточку/форму по отдельности.

## Ключевые ценности для пользователя

- экономия времени при рутинных правках 
- снижение вероятности ошибок при одинаковых действиях
- единообразное применение правил к группе

## Где находится

- Основной путь: Страница списка объектов → множественный выбор → кнопка «Настройки группы» / «Изменить выбранное»
- Альтернативные пути:
  - контекстное меню правой кнопкой мыши на выделенных строках
  - toolbar над таблицей после выбора ≥ 1 элемента
  
  Таблица должна содержать следующие столбцы:
  Object name
  URL объекта
  Группа доступа (группа пользователей имеющих права на редактирование или просмотр)
  Tag (каждый из объектов можно протегировать
  
  перед строкой должен быть селектор для выбора 
  
  Над таблицей должны быть указаны кнопки действий с дропдауном) (Изменить группу (со списком имеющихся групп и множественным выбором и кнопкой OK) , удалить ( после этого действия подтверждение).
  У объекта может быть несклько групп и несколько тегов, дропдаун должен содержать множественный выбор.
