# 104 - Описание страницы settings_users


Данная страница необходима для редактирования  пользователей

ВИзуально страница похожа на страницу управления объектами

Есть возможность добавить нового пользователя

Есть таблица со списком пользователей и селекторами слева для выполнения групповых операций.
Мы можем добавить пользователей в любую из групп (Administrators, Operators, Engineers, Guests). Для этого есть дропдаут и кнопки сбросить и применить по-аналогии со страницей настройки объектов.
